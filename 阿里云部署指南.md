# 自媒体项目 - 阿里云部署指南

## 📋 部署架构

```
用户 → 阿里云 ECS (公网 IP) → Nginx (反向代理) → Node.js 后端 (PM2) + Next.js 前端
                              ↓
                         MongoDB (本地或云数据库)
```

---

## 一、服务器准备

### 1.1 购买阿里云 ECS

**推荐配置**：
- **CPU**: 2 核
- **内存**: 4GB
- **系统**: Ubuntu 22.04 LTS 或 CentOS 7.9
- **带宽**: 3-5Mbps (根据访问量调整)
- **存储**: 40GB SSD 起步

### 1.2 安全组配置

在阿里云控制台开放以下端口：

| 端口 | 用途 | 协议 |
|------|------|------|
| 22 | SSH 连接 | TCP |
| 80 | HTTP | TCP |
| 443 | HTTPS | TCP |
| 27017 | MongoDB (仅内网访问) | TCP |

**安全组规则**：
```
入方向:
- 22/22 TCP 0.0.0.0/0 (SSH)
- 80/80 TCP 0.0.0.0/0 (HTTP)
- 443/443 TCP 0.0.0.0/0 (HTTPS)
- 27017/27017 TCP 127.0.0.1/32 (仅本地访问 MongoDB)
```

## 二、服务器环境配置

### 2.1 连接服务器

```bash
ssh root@<你的服务器公网 IP>
```

### 2.2 安装 Node.js

```bash
# Ubuntu/Debian
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs

# 验证安装
node -v  # 应显示 v20.x
npm -v   # 应显示 10.x
```

### 2.3 安装 MongoDB

```bash
# 导入 MongoDB 公钥
curl -fsSL https://www.mongodb.org/static/pgp/server-7.0.asc | sudo gpg -o /usr/share/keyrings/mongodb-server-7.0.gpg --dearmor

# 添加 MongoDB 源
echo "deb [ arch=amd64,arm64 signed-by=/usr/share/keyrings/mongodb-server-7.0.gpg ] https://repo.mongodb.org/apt/ubuntu jammy/mongodb-org/7.0 multiverse" | sudo tee /etc/apt/sources.list.d/mongodb-org-7.0.list

# 安装 MongoDB
sudo apt-get update
sudo apt-get install -y mongodb-org

# 启动 MongoDB
sudo systemctl start mongod
sudo systemctl enable mongod

# 检查状态
sudo systemctl status mongod
```

### 2.4 安装 PM2 (进程管理)

```bash
sudo npm install -g pm2
```

### 2.5 安装 Nginx (反向代理)

```bash
sudo apt-get install -y nginx
sudo systemctl start nginx
sudo systemctl enable nginx
```

### 2.6 安装 Git

```bash
sudo apt-get install -y git
```

## 三、项目部署

### 3.1 克隆项目

```bash
# 创建应用目录
sudo mkdir -p /var/www/selfmedia-app
sudo chown -R $USER:$USER /var/www/selfmedia-app
cd /var/www/selfmedia-app

# 克隆项目
git clone https://github.com/xiangqinburuouyu/selfmedia-app.git .
```

### 3.2 配置后端

```bash
cd /var/www/selfmedia-app/backend

# 安装依赖
npm install --production

# 创建生产环境配置
cat > .env << EOF
# 服务器配置
PORT=3000
NODE_ENV=production

# MongoDB 配置
MONGODB_URI=mongodb://localhost:27017/selfmedia-app

# JWT 配置 (生产环境务必修改！)
JWT_SECRET=your-super-secret-key-change-this-in-production-$(openssl rand -hex 32)
JWT_EXPIRE=7d

# 文件上传配置
UPLOAD_PATH=./uploads
MAX_FILE_SIZE=5242880
EOF
```

### 3.3 配置前端

```bash
cd /var/www/selfmedia-app/web

# 安装依赖
npm install

# 构建生产版本
npm run build
```

### 3.4 启动后端服务

```bash
cd /var/www/selfmedia-app/backend

# 使用 PM2 启动
pm2 start server.js --name selfmedia-backend

# 设置开机自启
pm2 startup
pm2 save
```

### 3.5 启动前端服务

```bash
cd /var/www/selfmedia-app/web

# 使用 PM2 启动 Next.js
pm2 start npm --name selfmedia-web -- start

# 保存 PM2 配置
pm2 save
```

## 四、Nginx 反向代理配置

### 4.1 创建 Nginx 配置文件

```bash
sudo nano /etc/nginx/sites-available/selfmedia-app
```

### 4.2 添加以下配置

```nginx
server {
    listen 80;
    server_name your-domain.com www.your-domain.com;  # 替换为你的域名

    # 前端 (Next.js)
    location / {
        proxy_pass http://localhost:3001;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    }

    # 后端 API
    location /api {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    }

    # 静态资源
    location /uploads {
        proxy_pass http://localhost:3000/uploads;
        proxy_http_version 1.1;
        proxy_set_header Host $host;
    }
}
```

### 4.3 启用配置

```bash
# 创建软链接
sudo ln -s /etc/nginx/sites-available/selfmedia-app /etc/nginx/sites-enabled/

# 删除默认配置 (可选)
sudo rm /etc/nginx/sites-enabled/default

# 测试配置
sudo nginx -t

# 重启 Nginx
sudo systemctl restart nginx
```

## 五、MongoDB 配置优化

### 5.1 限制 MongoDB 仅本地访问

```bash
sudo nano /etc/mongod.conf
```

确保配置：
```yaml
net:
  port: 27017
  bindIp: 127.0.0.1  # 仅允许本地访问
```

### 5.2 重启 MongoDB

```bash
sudo systemctl restart mongod
```

## 八、数据库备份

### 8.1 手动备份

```bash
# 备份 MongoDB
mongodump --out /var/backups/mongodb/$(date +%Y%m%d)
```

### 8.2 自动备份脚本

```bash
sudo nano /usr/local/bin/backup-mongodb.sh
```

```bash
#!/bin/bash
BACKUP_DIR="/var/backups/mongodb"
DATE=$(date +%Y%m%d_%H%M%S)
mkdir -p $BACKUP_DIR
mongodump --out $BACKUP_DIR/$DATE
# 保留最近 7 天的备份
find $BACKUP_DIR -type d -mtime +7 -exec rm -rf {} \;
```

```bash
# 添加执行权限
sudo chmod +x /usr/local/bin/backup-mongodb.sh

# 添加到 crontab (每天凌晨 2 点备份)
sudo crontab -e
# 添加：0 2 * * * /usr/local/bin/backup-mongodb.sh
```

## 快速部署脚本

创建 `deploy.sh` 一键部署：

```bash
#!/bin/bash
set -e

echo "🚀 开始部署自媒体项目..."

# 1. 更新系统
sudo apt-get update && sudo apt-get upgrade -y

# 2. 安装依赖
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs nginx git

# 3. 安装 MongoDB
curl -fsSL https://www.mongodb.org/static/pgp/server-7.0.asc | sudo gpg -o /usr/share/keyrings/mongodb-server-7.0.gpg --dearmor
echo "deb [ arch=amd64,arm64 signed-by=/usr/share/keyrings/mongodb-server-7.0.gpg ] https://repo.mongodb.org/apt/ubuntu jammy/mongodb-org/7.0 multiverse" | sudo tee /etc/apt/sources.list.d/mongodb-org-7.0.list
sudo apt-get update
sudo apt-get install -y mongodb-org
sudo systemctl start mongod
sudo systemctl enable mongod

# 4. 安装 PM2
sudo npm install -g pm2

# 5. 克隆项目
sudo mkdir -p /var/www/selfmedia-app
sudo chown -R $USER:$USER /var/www/selfmedia-app
cd /var/www/selfmedia-app
git clone https://github.com/xiangqinburuouyu/selfmedia-app.git .

# 6. 配置后端
cd backend
npm install --production
cp .env.example .env  # 或手动创建 .env

# 7. 配置前端
cd ../web
npm install
npm run build

# 8. 启动服务
cd ../backend
pm2 start server.js --name selfmedia-backend
cd ../web
pm2 start npm --name selfmedia-web -- start
pm2 save

# 9. 配置 Nginx
sudo ln -s /etc/nginx/sites-available/selfmedia-app /etc/nginx/sites-enabled/ 2>/dev/null || true
sudo nginx -t
sudo systemctl restart nginx

echo "✅ 部署完成！"
echo "📊 查看服务状态：pm2 status"
echo "📝 查看日志：pm2 logs"
```
