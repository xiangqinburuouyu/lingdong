## 技术栈详情

### 前端 (React Native + Expo)
| 类别 | 技术/库 |
|------|--------|
| 框架 | React Native (Expo SDK 50+) |
| 导航 | React Navigation 6.x |
| 状态管理 | Zustand (轻量) 或 Redux Toolkit |
| UI 组件 | React Native Paper / NativeBase |
| HTTP 请求 | Axios |
| 本地存储 | AsyncStorage |
| 图片处理 | expo-image |

### 后端 (Node.js)
| 类别 | 技术/库 |
|------|--------|
| 运行时 | Node.js 18+ |
| Web 框架 | Express 4.x |
| 数据库 | MongoDB + Mongoose |
| 认证 | JWT (jsonwebtoken) |
| 密码加密 | bcryptjs |
| 验证 | Joi / express-validator |
| CORS | cors |
| 文件上传 | multer |

### 开发工具
| 用途 | 工具 |
|------|------|
| 代码管理 | Git + GitHub |
| API 测试 | Postman / Insomnia |
| 数据库管理 | MongoDB Compass |
| 设计原型 | Figma (我可出设计稿) |

## 核心 API 设计

### 用户相关
```
POST   /api/auth/register     # 注册
POST   /api/auth/login        # 登录
POST   /api/auth/logout       # 登出
GET    /api/users/:id         # 获取用户信息
PUT    /api/users/:id         # 更新用户信息
```

### 文章相关
```
GET    /api/articles          # 获取文章列表
GET    /api/articles/:id      # 获取文章详情
POST   /api/articles          # 创建文章 (需认证)
PUT    /api/articles/:id      # 更新文章 (需认证)
DELETE /api/articles/:id      # 删除文章 (需认证)
GET    /api/categories        # 获取分类列表
```

### 互动相关
```
POST   /api/articles/:id/like     # 点赞
POST   /api/articles/:id/bookmark # 收藏
POST   /api/articles/:id/comment  # 发表评论
GET    /api/articles/:id/comments # 获取评论列表
```

### 搜索
```
GET    /api/search?q=关键词      # 搜索文章
```

## UI 设计规范（参考钛媒体）

### 颜色方案

```javascript
  primary: '#0066FF',      // 主色调 (钛媒体蓝)
```

```javascript
const typography = {
```

### 核心组件
1. **文章卡片** - 封面图 + 标题 + 摘要 + 作者 + 阅读量 + 时间
2. **底部导航** - 首页、分类、发现、我的 (4 个 tab)
3. **顶部搜索栏** - 搜索框 + 扫描/消息图标
4. **评论组件** - 头像 + 昵称 + 内容 + 时间 + 回复
