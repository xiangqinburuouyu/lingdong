# 自媒体博客 APP - 启动指南

## 📋 前提条件

确保已安装以下软件：

- **Node.js** >= 18.x ([下载](https://nodejs.org/))
- **MongoDB** >= 6.x ([下载](https://www.mongodb.com/try/download/community))
- **npm** 或 **yarn**（随 Node.js 安装）
- **Expo CLI**（前端开发工具）
- **Expo Go APP**（手机扫码运行）

---

## 🚀 快速启动（3 步）

### 步骤 1: 启动 MongoDB

**方式 A: 使用本地 MongoDB**

```bash
# 创建数据目录
mkdir -p /data/db

# 启动 MongoDB
mongod --dbpath /data/db
```

**方式 B: 使用 Docker**

```bash
docker run -d -p 27017:27017 --name mongodb mongo:latest
```

MongoDB 将在 `mongodb://localhost:27017` 运行。

---

### 步骤 2: 启动后端服务

```bash
# 进入后端目录
cd /home/admin/openclaw/workspace/temp/自媒体 APP/backend

# 安装依赖
npm install

# 配置环境变量
cp .env.example .env

# 编辑 .env 文件（可选，默认配置即可）
# MONGODB_URI=mongodb://localhost:27017/selfmedia-app

# 启动开发服务器
npm run dev
```

✅ 成功后看到：
```
✅ MongoDB 连接成功
🚀 服务器运行在 http://localhost:3000
📱 环境：development
```

---

### 步骤 3: 启动前端 APP

```bash
# 进入前端目录
cd /home/admin/openclaw/workspace/temp/自媒体 APP/mobile-app

# 安装依赖
npm install

# 修改 API 地址（重要！）
# 编辑 src/config/api.js
# 将 API_BASE_URL 改为你的后端地址
# const API_BASE_URL = 'http://localhost:3000/api';

# 启动 Expo 开发服务器
npx expo start
```

✅ 成功后会显示二维码，使用 **Expo Go APP** 扫描即可在手机上运行。

## 📱 在设备上运行

### iOS (iPhone)

1. 在 App Store 下载 **Expo Go**
2. 确保手机和电脑在同一 WiFi 网络
3. 打开 Expo Go，扫描终端中的二维码
4. APP 将自动加载并运行

### Android

1. 在 Google Play 下载 **Expo Go**
2. 确保手机和电脑在同一 WiFi 网络
3. 打开 Expo Go，扫描终端中的二维码
4. APP 将自动加载并运行

**Android 模拟器:**

## 🔧 常见问题

### Q1: MongoDB 连接失败？

**错误**: `MongoServerError: connect ECONNREFUSED`

**解决**:
```bash
# 检查 MongoDB 是否运行
ps aux | grep mongod

# 如果没有运行，启动它
mongod --dbpath /data/db
```

### Q2: 前端无法连接后端？

**错误**: `Network Error`

**解决**:
1. 确保后端服务正在运行（访问 http://localhost:3000/health）
2. 真机调试时，将 API 地址改为电脑 IP：
   ```javascript
   // src/config/api.js
   const API_BASE_URL = 'http://192.168.x.x:3000/api';
   ```
3. 确保防火墙允许 3000 端口访问

### Q3: npm install 失败？

**解决**:
```bash
# 清除缓存
npm cache clean --force

# 删除 node_modules 和 package-lock.json
rm -rf node_modules package-lock.json

# 重新安装
npm install
```

### Q4: Expo 扫码失败？

**解决**:
1. 确保手机和电脑在同一 WiFi 网络
2. 检查防火墙设置
3. 尝试使用隧道模式：
   ```bash
   npx expo start --tunnel
   ```

## 🧪 测试 API

### 使用浏览器

访问健康检查接口：
```
http://localhost:3000/health
```

应返回：
```json
{
  "status": "ok",
  "timestamp": "2026-03-21T05:44:00.000Z"
}
```

### 使用 curl

```bash
# 注册
curl -X POST http://localhost:3000/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{
    "username": "testuser",
    "email": "test@example.com",
    "password": "123456"
  }'

# 登录
curl -X POST http://localhost:3000/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "email": "test@example.com",
    "password": "123456"
  }'

# 获取文章列表
curl http://localhost:3000/api/articles
```
