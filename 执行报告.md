## 🆘 故障排查

### 问题 1：上传失败

**错误**: "403 Forbidden"

**解决**:
```bash
# 确保 token 有 workflow 权限
# 访问 https://github.com/settings/tokens
# 创建新 token，勾选 repo + workflow
```

### 问题 2：Secret 未找到

**错误**: "EXPO_TOKEN not found"

**解决**:
1. 确认 Secret 名称是 `EXPO_TOKEN`（大写）
2. 确认添加到正确的仓库
3. 重新添加 Secret

### 问题 3：构建失败

**错误**: "Gradle build failed"

**解决**:
- Expo 云端问题，等待 5 分钟后重试
- 或切换到本地构建方案
