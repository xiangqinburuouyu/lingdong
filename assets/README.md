# 资源文件说明

## 需要准备的图标文件

### 1. 应用图标 (icon.png)
- 尺寸：1024x1024 像素
- 格式：PNG
- 用途：应用商店图标

### 2. 自适应图标 (adaptive-icon.png)
- 尺寸：1024x1024 像素
- 格式：PNG
- 用途：Android 自适应图标

### 3. 启动图 (splash.png)
- 尺寸：1242x2436 像素（推荐）
- 格式：PNG
- 用途：应用启动画面

### 4. 网页图标 (favicon.png)
- 尺寸：48x48 像素
- 格式：PNG
- 用途：Web 版图标

## 推荐图标设计

### 自媒体博客 APP 图标建议
- **主色调**: #0066FF (钛媒体蓝)
- **元素**: 书本/文档/笔 等媒体相关图标
- **风格**: 简洁、现代、易识别

### 示例设计思路
```
┌─────────────────┐
│                 │
│    📱 或 📰     │
│    蓝色背景     │
│                 │
└─────────────────┘
```
