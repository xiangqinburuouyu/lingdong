## 🚀 下一步建议

### 方案 A：安装 MongoDB 后测试

```bash
# 安装 MongoDB
sudo apt-get install -y mongodb

# 启动 MongoDB
sudo systemctl start mongod

# 启动后端
cd backend
npm run dev

# 启动前端
cd mobile-app
npx expo start
```

### 方案 B：使用 MongoDB Atlas

1. 访问 https://www.mongodb.com/cloud/atlas
2. 创建免费集群
3. 获取连接字符串
4. 更新 `.env` 文件
5. 重启后端

### 方案 C：直接 APK 构建

1. 访问 https://expo.dev
2. 登录账号

5. 构建 APK
