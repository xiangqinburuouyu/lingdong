## 🚀 快速开始

### 1. 本地测试

**前提**: 需要安装 MongoDB

```bash
# 方式 A: 使用启动脚本
cd /home/admin/openclaw/workspace/temp/自媒体 APP
bash 启动脚本.sh

# 方式 B: 手动启动
# 启动 MongoDB
mongod --dbpath /data/db

# 启动后端
cd backend
npm run dev

# 启动前端
cd mobile-app
npx expo start
```

### 2. APK 构建

**方式 A: Expo 网页构建（推荐）**
1. 访问 https://expo.dev
2. 登录账号（qini0113@163.com）
3. 创建项目 `selfmedia-app`
4. 上传代码或连接 GitHub
5. 点击 Build → Android
6. 等待 10-20 分钟
7. 下载 APK

**方式 B: 本地构建**
```bash
cd mobile-app
npx expo run:android --variant debug
```
