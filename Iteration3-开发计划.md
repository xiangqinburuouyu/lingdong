## 🔧 技术方案

### 分享功能
- **库**: `expo-sharing` + `expo-share-intent`
- **实现**: 调用系统分享 + 自定义分享面板
- **海报**: `react-native-view-shot` 截图分享

### 推送通知
- **库**: `expo-notifications`
- **服务**: Expo Push Service
- **后端**: 存储 device token，发送通知

### 打赏功能
- **微信支付**: 需要商户号，H5 支付
- **支付宝**: 需要商户号，手机网站支付
- **替代方案**: 先实现模拟打赏，后续接入真实支付

### 数据分析
- **后端**: MongoDB 聚合查询
- **统计**: 阅读量、点赞、评论、收藏
- **前端**: 图表展示（可选 `react-native-chart-kit`）

### 深色模式
- **库**: `@react-native-community/dark-mode` 或自定义
- **存储**: AsyncStorage 保存主题偏好
- **适配**: 所有组件支持主题切换

## 📝 开发步骤

### Phase 3.1: 分享功能
1. 安装 `expo-sharing` 和相关依赖
2. 创建分享组件（ShareModal）
3. 实现分享功能（微信、微博、QQ、复制链接）
4. 在文章详情页添加分享按钮
5. 测试分享功能

### Phase 3.2: 推送通知
1. 安装 `expo-notifications`
2. 配置通知权限
3. 获取并保存 device token
4. 后端实现通知发送 API
5. 创建通知中心页面
6. 测试通知推送

### Phase 3.3: 数据分析
1. 后端实现统计 API
2. 创建数据统计模型
3. 前端实现数据展示页面
4. 添加图表可视化

### Phase 3.4: 打赏功能
1. 后端实现打赏 API
2. 创建打赏记录模型
3. 前端实现打赏 UI
4. 接入支付 SDK（或先模拟）

### Phase 3.5: 深色模式
1. 创建主题配置
2. 实现主题切换逻辑
3. 适配所有组件
4. 添加设置入口
